# Крестики-нолики

## Описание игры
- Игровое поле 3×3.
- Два игрока по очереди ставят X и O.
- Побеждает тот, кто первым выстроит три своих символа в ряд (горизонталь, вертикаль или диагональ).
- Если все клетки заполнены, но победителя нет – объявляется ничья.

## Логика победы/поражения
- Проверка всех выигрышных комбинаций после каждого хода.
- При обнаружении комбинации игра останавливается, выводится сообщение о победителе.
- Если все клетки заполнены и нет победителя – ничья.

## Особенности реализации
- Игра написана на чистом JavaScript без использования сторонних библиотек.
- Интерфейс адаптивен, игровое поле центрировано.
- Есть кнопка перезапуска игры.

## Игровая механика
- Победитель определяется перебором всех 8 выигрышных линий.
- Клики обрабатываются через событие `click`, привязанное к каждой клетке.
- Перезапуск сбрасывает массив `board`, переменную текущего игрока и очищает клетки.

## Структура данных
- `board` – массив из 9 строк, где каждый элемент хранит 'X', 'O' или '' (пусто).
- `winPatterns` – массив из 8 выигрышных комбинаций (индексов клеток).
- `gameActive` – флаг, показывающий, продолжается ли игра.

## Алгоритмы
- **Проверка победы:** проход по `winPatterns`, проверка равенства всех трёх клеток одному символу.
- **Определение ничьи:** если после проверки победы ни одна комбинация не сработала, но в `board` нет пустых ячеек.
- **Смена игрока:** после успешного хода (если игра не закончена) `currentPlayer` меняется на противоположный.
